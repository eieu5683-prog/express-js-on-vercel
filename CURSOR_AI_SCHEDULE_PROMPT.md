# Cursor AI 프롬프트: 일정 데이터 변환

엑셀 데이터를 `src/data/scheduleData.ts`의 `GRANT_SCHEDULE` 배열로 변환하는 프롬프트입니다.

## 기본 프롬프트

```
나랏돈네비의 src/data/scheduleData.ts 파일을 업데이트해줘. 
내가 아래에 제공하는 엑셀 데이터를 바탕으로 GRANT_SCHEDULE 상수 배열을 생성해줘.
```

## 상세 프롬프트 (데이터 변환 규칙 포함)

```
나랏돈네비의 src/data/scheduleData.ts 파일을 업데이트해줘. 
내가 아래에 제공하는 엑셀 데이터를 바탕으로 GRANT_SCHEDULE 상수 배열을 생성해줘.

[데이터 변환 규칙]

1. ID 생성: '연도-월-순번' 형식으로 고유한 ID를 부여해줘.
   예시: '2026-01-1', '2026-01-2', '2026-02-1'

2. 등급 매칭:
   - 'IT/소프트웨어', '바이오', 'AI', '로봇' 관련 내용이 있으면 targetGrade에 ['S', 'A']를 포함시켜줘
   - '일반 소상공인', '소상공인', '전통시장' 관련 내용이 있으면 ['B', 'C']를 포함시켜줘
   - 모든 업종 대상이면 ['S', 'A', 'B', 'C', 'D']를 포함시켜줘
   - 세무 일정은 모든 등급 ['S', 'A', 'B', 'C', 'D']를 포함시켜줘

3. 세무 일정 포함:
   - 부가가치세, 소득세, 원천징수, 연말정산 등 세무 관련 내용은 category: 'TAX'로 분류해줘
   - 지원금, 공고, 패키지 등은 category: 'GRANT'로 분류해줘

4. 정렬: 연도와 월 순서대로 오름차순 정렬해줘

5. 법규 준수: description 하단에 반드시 다음 문구를 포함해줘:
   "본 요약은 나랏돈네비 AI 기술로 작성되었습니다."

6. 날짜 형식: startDate와 endDate는 'YYYY-MM-DD' 형식으로 변환해줘

7. 데이터 타입: src/data/types.ts에 정의된 GrantSchedule 인터페이스를 엄격히 따라야 해

[엑셀 데이터 형식 예시]
연도 | 월 | 사업명 | 대상 | 내용 | 시작일 | 종료일 | 지원금액 | 링크
2026 | 1 | 예비창업패키지 | IT/소프트웨어 | 최대 1억원 지원... | 2026-01-15 | 2026-02-28 | 최대 1억원 | https://...
```

## 사용 방법

### 1단계: 엑셀 데이터 준비

엑셀 시트에서 다음 컬럼을 포함한 데이터를 복사:
- 연도
- 월
- 사업명/제목
- 대상 (IT/소프트웨어, 일반 소상공인 등)
- 내용/설명
- 시작일
- 종료일
- 지원금액 (선택사항)
- 링크 (선택사항)

### 2단계: Cursor AI 채팅창 열기

- `Cmd + L` (Mac) 또는 `Ctrl + L` (Windows)
- 또는 Cursor AI 채팅 아이콘 클릭

### 3단계: 프롬프트 입력

위의 "상세 프롬프트"를 복사하여 붙여넣기

### 4단계: 엑셀 데이터 붙여넣기

복사한 엑셀 데이터를 프롬프트 아래에 붙여넣기

### 5단계: 결과 확인

`src/data/scheduleData.ts` 파일의 `GRANT_SCHEDULE` 배열이 업데이트되었는지 확인

## 문제 해결

### 변환이 제대로 안 될 때

1. **엑셀 데이터 형식 확인**
   - 컬럼 순서가 올바른지 확인
   - 구분자가 일관된지 확인 (탭 또는 |)

2. **프롬프트를 더 구체적으로 작성**
   - 데이터 형식 예시 포함
   - 변환 규칙 명시

3. **단계별 변환**
   - 먼저 기본 정보만 변환
   - 그 다음 등급 매칭
   - 마지막으로 AI 표기 문구 추가

### 타입 오류 발생 시

1. `src/data/types.ts`의 `GrantSchedule` 인터페이스 확인
2. 필수 필드가 모두 포함되었는지 확인
3. 타입이 올바른지 확인 (year: number, month: number 등)

## 주의사항

- **ID 중복 방지**: 같은 연도-월에 여러 일정이 있으면 순번을 다르게 부여
- **등급 매칭 정확성**: 대상 업종에 따라 정확한 등급 매칭
- **AI 표기법**: description 하단에 반드시 포함
- **정렬**: 연도와 월 순서대로 정렬하여 사용자 경험 개선
