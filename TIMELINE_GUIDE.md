# 타임라인 페이지 가이드

2026년 월별 지원금 & 세무 타임라인 UI 사용 가이드입니다.

## 개요

타임라인 페이지는 `src/data/scheduleData.ts`에 저장된 `GRANT_SCHEDULE` 데이터를 자동으로 불러와 현재 날짜를 기준으로 가장 가까운 일정을 강조하여 보여줍니다.

## 주요 기능

### 1. 연도별 필터링
- 2026, 2027, 2028년 데이터를 탭으로 전환
- 기본값: 2026년

### 2. 월별 그룹화
- 12개월을 세로형 타임라인으로 표시
- 각 월별로 지원금 및 세무 일정을 카드 형태로 표시

### 3. 시각적 구분
- **세무 일정 (TAX)**: 호박색(Amber) 테두리 및 배경
- **정부 지원금 (GRANT)**: 파란색(Blue) 테두리 및 배경

### 4. 현재 월 강조
- 현재 월의 일정은 녹색 링과 펄스 애니메이션으로 강조
- 과거 월: 회색 배지
- 현재 월: 녹색 배지 (펄스 애니메이션)
- 미래 월: 파란색 배지

### 5. 호버 효과
- framer-motion을 사용한 부드러운 호버 애니메이션
- 카드가 오른쪽으로 살짝 이동

## 데이터 구조

타임라인은 `GRANT_SCHEDULE` 배열에서 다음 필드를 사용합니다:

- `id`: 고유 ID
- `year`: 연도
- `month`: 월
- `title`: 사업명
- `category`: 'GRANT' 또는 'TAX'
- `description`: 주요 혜택 및 내용
- `targetGrade`: 지원 가능 등급
- `startDate`: 시작일 (선택사항)
- `endDate`: 종료일 (선택사항)
- `amount`: 지원금액 (선택사항)
- `link`: 링크 (선택사항)

## 사용 방법

### 1. 페이지 접속
```
/timeline
```

### 2. 연도 선택
- 상단 탭에서 2026, 2027, 2028년 중 선택
- 기본값: 2026년

### 3. 일정 확인
- 월별로 그룹화된 일정 확인
- 현재 월은 녹색으로 강조
- 카드에 마우스를 올리면 호버 효과

### 4. 상세 정보
- 각 카드에 일정 제목, 설명, 등급, 링크 표시
- 링크가 있는 경우 "자세히 보기 →" 버튼 클릭

## 모바일 최적화

## 데이터 업데이트

타임라인에 표시되는 데이터를 업데이트하려면:

1. `src/data/scheduleData.ts`의 `GRANT_SCHEDULE` 배열 수정
2. Cursor AI를 사용하여 엑셀 데이터 변환 (참고: `CURSOR_AI_SCHEDULE_PROMPT.md`)
3. 페이지 새로고침하여 변경사항 확인

## 문제 해결

### 데이터가 표시되지 않을 때
- `GRANT_SCHEDULE` 배열에 해당 연도의 데이터가 있는지 확인
- 브라우저 콘솔에서 오류 확인

### 스타일이 적용되지 않을 때
- Tailwind CSS가 제대로 설정되었는지 확인
- `app/globals.css`에 `@tailwind` 디렉티브가 포함되어 있는지 확인
- 개발 서버 재시작

### 애니메이션이 작동하지 않을 때
- framer-motion이 설치되었는지 확인: `npm list framer-motion`
- 브라우저 콘솔에서 오류 확인
