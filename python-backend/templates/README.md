# HWP 템플릿 디렉토리

## 사용 방법

1. 정부 표준 사업계획서 HWP 양식 파일을 준비합니다.
2. 파일명을 `psst_template.hwp`로 저장합니다.
3. 이 디렉토리에 배치합니다.

## 템플릿 필드 매핑

템플릿 파일의 특정 위치에 PSST 데이터가 자동으로 삽입됩니다.

### 필드 매핑 규칙

- **제목**: "PSST 사업계획서"
- **업종 정보**: 메타데이터에서 자동 추출
- **Problem 섹션**: 문제 인식 내용
- **Solution 섹션**: 해결 방안 내용
- **Scale-up 섹션**: 성장 전략 내용
- **Team 섹션**: 팀 구성 내용
- **AI 표기 문구**: 문서 하단에 자동 추가

## 커스터마이징

`app/services/hwp_generator.py` 파일의 `_insert_psst_data_to_hwp` 또는 `_insert_psst_data_to_docx` 함수를 수정하여 템플릿 필드 매핑을 커스터마이징할 수 있습니다.
